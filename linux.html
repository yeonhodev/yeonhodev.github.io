Linux

  • Comand lines
    • pwd; 현재 위치(디렉토리) 확인
    • mkdir; 새로운 디렉토리(폴더) 생성
    • ls -al; 현재 위치에서 폴더/파일 자세한 정보 보여주기
  • Tips
    • Command line 앞에 프로그램명 없이 현재 위치에서 프로그램 실행하는 방법
      • ./helloworld.py 라고 Terminal에 타이핑하면 permission denied라는 메세지가 나옴
      • li- al
      • sudo chmod a+x helloworld.py -> 모든 사용자에게 실행 권한을 부여하는 명령어
      • 파일을 읽지 못하는 에러가 발생할 경우 .py 파일 제일 윗쪽에 다음의 내용을 추가(Mac 기준) -> #!/usr/bin/env python3
      • 혹시 경로 이상으로 실행이 되지 않는 경우에는 -> Terminal에서 type python3라고 입력하고 나오는 경로를 .py 파일 윗 쪽에 추가하여 실행 시킬 수 있음
